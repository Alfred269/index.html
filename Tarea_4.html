¿Qué lenguajes de programación contaminan más?

Considerando criterios básicos podemos comprender que el lenguaje que mas contamina es aquel que mas recursos requiere para ser ejecutado, por lo tanto mas energía consumirá de el ambiente. Un claro ejemplo de este seria el lenguaje de programación Perl seguidos del lenguaje Python y el lenguaje Ruby. Python es muy comprensible que utilice mucha energía ya que tiene diversos servidores a su poder, como podría ser el Google Colab que se ejecuta por un navegador web y es capas de ejecutar el lenguaje Python sin ningún problema y sin instalar nada.

El lenguaje Ruby se utiliza para diferentes ámbitos el principal podría ser el desarrollo de aplicaciones web esto genera cantidades de energía fuera de un promedio estándar aparte de esto este lenguaje también es utilizado por los programadores para el desarrollo de condominios virtuales.

Al contrario de esto los lenguajes mas seguros para utilizar serian el Lenguaje de programación C y su variación C++, el lenguaje Ruby y el lenguaje Ada ¿Y porque resulta ser así? Pues estos lenguajes son usados principalmente para el desarrollo de sistemas operativos y gracias a esto se ha comprobado que la energía necesaria para ser usados correctamente basta y sobra con la que le proporciona la computadora en que se ejecute.
